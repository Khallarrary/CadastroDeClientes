# Cadastro de Clientes

Este é um aplicativo de console simples para cadastro e gerenciamento de clientes em C#. Ele permite adicionar, visualizar, editar e excluir clientes de uma lista armazenada em memória.

## Funcionalidades

-   📌 **Cadastrar clientes** (Nome, Idade e CPF)
-   🔍 **Visualizar clientes cadastrados**
-   ✏️ **Editar dados de um cliente** (Em desenvolvimento)
-   ❌ **Excluir um cliente** (Em desenvolvimento)
-   🚪 **Sair do programa**

## Requisitos

-   .NET SDK instalado (versão 5.0 ou superior)

## Como Executar o Projeto

1.  Clone este repositório:
    
    ```sh
    git clone https://github.com/Khallarrary/CadastroDeClientes.git
    
    ```
    
2.  Navegue até a pasta do projeto:
    
    ```sh
    cd CadastroDeClientes
    
    ```
    
3.  Compile e execute o programa:
    
    ```sh
    dotnet run
    
    ```
    

## Estrutura do Projeto
